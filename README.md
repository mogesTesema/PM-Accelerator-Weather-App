# PM-Accelerator-Weather-App
[![Backend CI/CD](https://github.com/mogesTesema/PM-Accelerator-Weather-App/actions/workflows/backend.yml/badge.svg)](https://github.com/mogesTesema/PM-Accelerator-Weather-App/actions/workflows/backend.yml)

Engineered a production-grade weather intelligence system for the PM Accelerator AI assessment. Built with a Django backend (full CRUD), React frontend, and Pinecone for semantic location retrieval. Integrates real-time APIs, automated multi-format exports, and contextual multimedia via YouTube and Google Maps.
